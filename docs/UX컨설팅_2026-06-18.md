# 벧엘구역 UX 컨설팅 설계서 (v2)
작성일: 2026-06-18 / 수정: 2026-06-18

## 1. 타겟 사용자 정의

### 핵심 전제
- 타겟은 **이미 신앙생활 중인 사람**. QT, 구역, 기도제목 등 교회 용어에 익숙함.
- 완전 초심자는 1차 타겟이 아님 → 용어 변경 불필요.
- 성경읽기·찬송가는 바이블애플 등 기존 앱이 충분히 커버 → 우리의 차별점이 아님.

### 페르소나 A: 신앙 습관 형성자 (메인 타겟)
- 교회 다니고 있음, QT·기도를 **더 체계적으로** 하고 싶음
- 매일 루틴을 앱으로 만들고 싶음 — 스트릭, 기록, 성장 확인
- 구역 모임이 있지만 앱을 개인적으로도 적극 활용
- 모바일 중심, 30~50대 주류

### 페르소나 B: 소모임 리더 (핵심 유입 채널)
- 구역장, 목장 리더, 셀 리더
- 지금은 카카오톡으로 공지·기도제목·출석 확인을 하고 있음
- **핵심 니즈:** 구성원 QT 현황 한눈에 보기, 독려, 공지, 성경공부 출제
- 리더가 앱을 쓰면 구성원 전체를 데려옴 → 최대 유입 채널
- 리더 경험이 좋으면 다른 구역에 추천 → 바이럴

### 페르소나 C: 기존 구역원 (현재 활성 사용자)
- 이미 앱을 매일 사용 중 → UX 변경 시 기존 동선 최대한 보존
- 구역장 안내에 따라 사용, 기능 탐색보다 습관적 사용

## 2. 현황 진단

### 2-1. 앱 구조 현황

| 영역 | 현재 상태 |
|------|-----------|
| 네비게이션 | 하단 탭 4개 (홈/기도/공부/QT) + 사이드바 8개 메뉴 |
| 홈(대시보드) | 인사말 → WeeklyGuide → Trial배너 → QT카드 → 통계4개 → 일정 → 기도제목 → 성경공부 |
| QT 플로우 | QTMain → QTPray → QTComplete (3단계) |
| UI 모드 | simple / full 분기 + 역할(master/leader/member) 분기 |
| 성경읽기 | 라이센스 보호로 simpleHide=true (설계 의도) |

### 2-2. 핵심 문제 진단

**문제 1 [P0 — 사용성]: "지금 뭘 해야 하나?" 불명확**
- 대시보드에 8개 섹션 동시 표시 → 시각적 과부하
- WeeklyGuideCard가 phase 1/2/3 × 역할 분기로 내부 복잡도 높음
- 신규 가입자(리더 포함)가 첫 행동을 스스로 결정하지 못함

**문제 2 [P0 — 리더 경험]: 소모임 리더를 위한 전용 허브가 없음**
- 구성원 QT 현황, 기도제목 등록 여부, 성경공부 답변 완료율을 한 화면에서 볼 수 없음
- 독려(깨우기) 기능 없음 → 지금도 카톡 병행 필요
- 리더가 앱에서 충분한 가치를 못 느끼면 구성원 유입도 없음

**문제 3 [P1 — 리텐션]: 개인 성장 트래킹이 스트릭 숫자 하나뿐**
- QT 연속 일수는 있으나 캘린더, 진도 바, 배지 등 시각적 성취감 없음
- "내 묵상 기록을 다시 읽고 싶다"는 니즈 미충족

**문제 4 [P1 — 구조]: 하단 탭이 기능 목록, 행동 목적 불명확**
- 홈 / 기도 / 공부 / QT → "기도를 하러 가는 건지, 올리러 가는 건지" 모름
- 성경읽기, 일정, 관리자 등은 사이드바에 숨겨져 발견 어려움
- "나"에 해당하는 탭(내 기록, 설정, 프로필)이 없어 사이드바가 비대해짐

**참고 — 성경읽기 simpleHide:**
- 라이센스 문제로 의도적 보호 → 구조적 버그 아님
- 개선 방향: 라이센스 있는 교회는 자동 활성화 (현재 hasModule 구조 유지)
- 신규 교회 온보딩 시 "성경 본문 라이센스 취득 후 활성화" 안내 명시

## 3. 경쟁사 벤치마킹

### 3-1. 바이블 애플

**강점:**
- 성경읽기 특화 — 17개 번역본, 음성 4개 국어
- 배지 시스템: 통독 완료 시 퀘스트처럼 보상
- 그룹통독: 함께 읽기, "깨우기" 푸시 알림
- 7일 미니통독 — 달성 가능한 단기 챌린지
- AI 개인화 알림

**우리와 차이:**
- 성경읽기 중심 앱 → QT·묵상·소모임 기능 없음
- 그룹은 통독 목적으로만 구성

**우리가 배울 점:**
- 깨우기 알림 (구성원 독려) → 리더 허브 기능으로 도입
- 배지/챌린지 시스템 → 개인 성장 트래킹 강화에 적용
- 단기 챌린지 단위 설계 (7일, 21일)

### 3-2. 큐티한스푼

**강점:**
- QT 플로우 명확: 말씀 → 해설 → 질문 → 기록
- 폰트 크기 조절, 배경 색상 커스터마이징
- 감사노트 (개인 기록)

**약점 (우리가 앞설 기회):**
- 소모임/그룹 기능 없음 → 우리의 핵심 차별점
- 구성원 활동 현황 없음
- 동기부여 요소 (배지, 스트릭 시각화) 부족

**우리가 배울 점:**
- QT 상세 화면의 플로우 단계 명확성
- 폰트 크기 조절 기능 (접근성)

### 3-3. 포지셔닝

```
개인 묵상 특화                         소모임/그룹 특화
큐티한스푼  ←━━━━━━━━━━━━━━━━━━━━━━━━━━→  [목표]
                                              ↑
                               개인 QT + 소모임 운영 통합
바이블애플 (성경읽기 중심, 그룹통독)
```

**우리의 핵심 차별점:**
> "개인 QT 습관 + 소모임 리더 도구"를 하나의 앱으로
> 리더가 쓰면 구성원 전체가 따라온다

## 4. UX 개선 방향

### P0-A: 홈 화면 단순화 — "오늘 한 가지"

**현재:** 8개 섹션 동시 노출
**개선:** QT 히어로 카드 1개 집중 + 구역 활동 요약

```
┌──────────────────────────────┐
│  벧엘구역    🔔  🔥 12일     │  ← 스트릭 헤더에 표시
├──────────────────────────────┤
│  TODAY · 6월 18일 (수)        │
│                              │
│  ┌────────────────────────┐  │
│  │  시편 23:1-6            │  │  ← QT 히어로 카드 (크게)
│  │  "여호와는 나의 목자시니" │  │
│  │                        │  │
│  │  [오늘의 QT 시작하기 →] │  │
│  └────────────────────────┘  │
│                              │
│  우리 구역                   │  ← 구역 활동 (간결하게)
│  ✓ 김민지  ✓ 이정호  ○ 박성훈│
│  기도제목 3개  [구역 보기 →] │
│                              │
│  다가오는 일정: 6/21 구역모임  │  ← 일정 1줄만
└──────────────────────────────┘

하단 탭: 홈 / QT / 구역 / 나
```

### P0-B: 소모임 리더 허브 신설

**현재:** 관리자 탭(AdminDashboard)이 있으나 구성원 현황 파악이 산재됨
**개선:** 리더 전용 "구역" 탭에 통합 대시보드 추가

```
[구역 탭 — 리더 화면]

구성원 QT 현황 (오늘)
  ✓ 김민지     ○ 이정호 (알림)     ✓ 박성훈
  ✓ 최지영     ○ 한민수 (알림)     ✓ 이수연
  완료 4 / 미완료 2
  [미완료 구성원에게 깨우기 보내기]

기도제목                  [+ 등록]
  • 박성훈 · 취업 기도 (3명 함께기도)
  • 최지영 · 건강 회복

이번 주 성경공부           [출제하기]
  답변 완료: 5/6명

다가오는 일정              [+ 등록]
  6/21 (토) 19:00 구역모임
```

**리더가 카톡 없이도 할 수 있어야 하는 것:**
- 구성원 QT 완료 현황 확인
- 미완료 구성원에게 한 번에 독려 알림 (깨우기)
- 기도제목 등록 및 공지
- 성경공부 출제 + 답변 현황 확인
- 일정 등록 + 공지

### P1: 하단 탭 재구성

**현재:** 홈 / 기도 / 공부 / QT (행동보다 기능 이름)
**개선:** 홈 / QT / 구역 / 나

```
홈         QT         구역          나
🏠        ✝️         👥           👤
오늘 현황   묵상하기    구역 활동     내 기록
```

**변경 근거:**
- QT: 탭 이름 그대로 유지 (타겟이 아는 용어)
- 구역: 기도+공부+일정+리더 기능 통합 → "구역"이라는 단어로 집약
- 나: 현재 사이드바에 흩어진 프로필/기록/설정을 한 곳에

**메뉴 항목 배치:**

| 탭 | 포함 기능 |
|----|-----------|
| 홈 | 오늘의 QT 히어로 + 구역 활동 요약 + 스트릭 |
| QT | QT 작성 + 기도 + 완료 + 지난 QT 보기 |
| 구역 | 기도제목 + 성경공부 + 일정 + (리더) 구성원 현황 + 깨우기 |
| 나 | 묵상 캘린더 + 스트릭 상세 + 배지 + 프로필 + 설정 |

→ 성경읽기: 라이센스 있는 교회는 "나" 탭 또는 "QT" 탭 하위에 노출

### P2: 개인 성장 트래킹 ("나" 탭 신설)

```
[나 탭]

OOO님
연속 12일 · 이번 달 18회

── 나의 QT 기록 ──
[6월 캘린더]
● ● ○ ● ● ● ○ ● ...  ← 완료일 색상 표시
[날짜 누르면 그날 묵상 열람]

── 배지 ──
✅ 첫 QT       ✅ 7일 연속
🔒 30일 연속   🔒 100일 연속
[배지 전체 보기]

── 설정 / 알림 / 프로필 / 문의 ──
```

### P3: 깨우기 알림 (Nudge)

- 기존 push-dispatch Edge Function 활용
- 리더가 구역 탭에서 "미완료 구성원에게 깨우기" 버튼 한 번
- 발송 내용: "오늘의 QT 함께해요 :) — {리더이름}이 기다리고 있어요"
- 하루 1회 제한 (과남용 방지)

## 5. 정보 구조 (IA) 재설계

### 현재 IA
```
앱
├── 대시보드 (과부하)
├── 오늘의 QT → QTPray → QTComplete
├── 구역성경공부 (목록/상세)
├── 주요일정
├── 기도제목 (내기도/중보)
├── 성경 (라이센스 보호)
├── 관리자 (leader+)          ← 기능 분산
└── 구역 관리 (master)        ← 기능 분산
```

### 개선 IA
```
앱
├── 홈
│   ├── QT 히어로 카드 (오늘)
│   ├── 구역 활동 요약
│   └── 스트릭 + 다음 일정
│
├── QT
│   ├── 오늘의 QT (말씀 + 묵상 + 기도)
│   ├── 지난 QT 목록
│   └── [라이센스 있는 교회] 성경읽기
│
├── 구역
│   ├── [리더] 구성원 현황 + 깨우기
│   ├── 기도제목 (내 기도 + 함께기도)
│   ├── 성경공부 (목록 + 답변)
│   └── 일정
│
├── 나
│   ├── QT 캘린더 (묵상 기록 열람)
│   ├── 배지 / 챌린지
│   ├── 프로필 / 알림 설정
│   └── 문의 / 로그아웃
│
└── [master] 구역 관리
    ├── 구성원 초대/관리
    └── 교회 설정
```

## 8. 디자인 시스템 방향

### 색상 체계
- **Primary (보라):** 현재 유지 — 앱 전체 브랜드
- **QT 섹션:** 따뜻한 앰버/오렌지 (현재 QT 카드 색상 계승)
- **구역 섹션:** 차분한 블루-그린 (공동체 느낌)
- **성취/배지:** 골드 + 그레이 (잠금 상태)

### 타이포그래피
- 말씀 본문: font-display (serif) 유지, **폰트 크기 조절 기능 추가**
- UI 레이블: sans-serif, 간결

### 개선 우선 컴포넌트
1. **홈 QT 히어로 카드** — 더 크고 임팩트 있게, 미완료/완료 두 상태 강화
2. **구성원 현황 아이템** — 아바타 + 이름 + 완료 여부 + 깨우기 버튼
3. **QT 캘린더** — 월간 뷰, 날짜별 색상 도트
4. **배지 컴포넌트** — 잠금/해제 상태, 호버 시 조건 표시
