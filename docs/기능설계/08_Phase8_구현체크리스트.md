# Phase 8 구현 체크리스트

## 1. 문서 목적

이 문서는 `벧엘구역`의 Phase 8 구현을 실제 작업 단위로 쪼개어 관리하기 위한 체크리스트다.

대상 범위:

- 설치형 앱 전환
- 웹푸시 구독/발송
- 추천 찬양 AI
- 수동 입력 기반 설교/예배 요약 AI

## 2. 실행 원칙

- 운영 중인 서비스이므로 작은 묶음 단위로 배포한다.
- 브랜드 자산과 설치 흐름을 먼저 안정화한다.
- 웹푸시는 앱 내 알림과 분리하지 않고 확장 구조로 붙인다.
- AI는 보조 기능으로 시작하고, 자동 발행은 나중으로 미룬다.

## 3. 구현 묶음
