# 운영 자동화 및 추가 요건

## 1. 프론트엔드 즉시 적용 가능 항목

### 1.1 카카오톡 공지 메시지 생성

- 관리자 대시보드에 공지 발송 기능을 추가한다.
- 일정, 성경공부, 성경 읽기, 기도 제목용 템플릿을 제공한다.
- 서비스 링크를 포함한 메시지를 생성하고 클립보드로 복사한다.
- 별도 저장 모델은 필요 없고 `navigator.clipboard.writeText()`만 사용한다.

### 1.2 성경 공부 답안 접근 권한

정책은 다음과 같다.

| 대상 | 답변 내용 | 완료 여부 |
|------|----------|----------|
| 본인 | 열람/편집 가능 | 표시 |
| 타 구역원 | 열람 불가 | 표시 |
| 구역장 | 열람 불가 | 표시 |

```typescript
StudyAnswer {
  studyId: string;
  userId: string;
  userName: string;
  answers: Record<number, string>;
  completed: boolean;
}
```

## 2. 운영 자동화 기능

### 2.1 오늘의 묵상 자동 수집 + AI 요약

- 대상: `https://sum.su.or.kr:8888/bible/today`
- 수집 항목: 성경 구절, 본문, 해설 탭, 오디오 링크, 묵상 링크
- 실행 시각: 매일 06:00 KST, 실패 시 06:30 재시도
- 출력: 300자 이내 AI 요약 + 적용 질문 1개
- 표시 위치: 대시보드 오늘의 QT 영역 대체

```typescript
DailyDevotional {
  id: string;
  date: string;
  scripture: string;
  scriptureText: string;
  commentary: string;
  aiSummary: string;
  audioUrl: string;
  devotionalUrl: string;
  scrapedAt: string;
}
```

### 2.2 주보 PDF 기반 성경공부 자동 생성

- 대상: 벧엘교회 주보 PDF 7페이지
- 실행 시각: 매주 일요일 20:00 KST
- 흐름: PDF 다운로드 -> 텍스트 추출 -> OpenAI 구조화 -> DB 등록 -> 구역장 검토/발행
- 마감 시각: 다음 주일 11:20 KST

```typescript
BibleStudy {
  id: string;
  weekNumber: number;
  studyPeriodStart: string;
  studyPeriodEnd: string;
  title: string;
  scripture: string;
  introduction: string;
  questions: string[];
  sourcePdfUrl: string;
  published: boolean;
  publishedAt: string;
  createdAt: string;
}
```

### 2.3 주간 운영 마감

매주 주일 11:20 KST에 다음 항목을 동시에 마감한다.

1. 성경공부 답변 잠금
2. 성경 읽기 주간 입력 잠금
3. 기도 제목 주차 마감
4. 출석 응답 잠금
5. 주간 보고서 집계

### 2.4 구역장 모임 보고서

- 조회 시점: 매주 주일 15:00 이후
- 주요 항목: 참석 인원, 참석/불참 명단, 성경 읽기 합계, 공부 완료율, 기도 제목 수
- 관리자 대시보드에 "주간 보고" 탭을 추가한다.
- 카카오톡 보고 메시지 복사와 PDF 저장 기능을 제공한다.

```typescript
WeeklyReport {
  id: string;
  weekNumber: number;
  year: number;
  periodStart: string;
  periodEnd: string;
  attendanceCount: number;
  attendees: string[];

  totalBibleChapters: number;
  bibleReadingByMember: {
    userId: string;
    userName: string;
    chapters: number;
  }[];
  studyCompletedCount: number;
  studyTotalCount: number;
  prayerRequestCount: number;
```
