# Core Features / 핵심 업무 기능

> Public document / 공개 문서
> Last reviewed / 마지막 점검: 2026-07-07

## English Summary

This document summarizes the main product workflows implemented in bethel803. The application is organized around a small group leader's weekly rhythm: prepare study material, care for members, review participation, receive prayer requests, and close the week with usable records.

## 한국어 요약

이 문서는 bethel803에 구현된 주요 업무 흐름을 정리합니다. 앱은 소그룹 리더의 반복 운영 흐름을 중심으로 구성됩니다. 성경공부를 준비하고, 구성원을 돌보고, 참여 상태를 확인하고, 기도제목을 받고, 주간 기록을 남기는 과정을 하나의 흐름으로 연결합니다.
