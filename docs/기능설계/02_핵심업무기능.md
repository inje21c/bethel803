# 핵심 업무 기능

## 1. 인증

**페이지:** `/login`, `/reset-password`, `/pending`

| 기능 | 설명 | 상태 |
|------|------|------|
| 로그인 | 이메일/비밀번호 기반 로그인 | 구현됨 |
| 회원가입 | 이름, 이메일, 비밀번호, 구역 선택 후 가입 | 구현됨 |
| 승인 대기 | 가입 후 `pending` 상태 사용자 대기 화면 | 구현됨 |
| 비밀번호 재설정 메일 | 재설정 이메일 발송 | 구현됨 |
| 비밀번호 변경 | 재설정 페이지 및 프로필에서 변경 | 구현됨 |
| 로그아웃 | 세션 종료 후 로그인 화면 이동 | 구현됨 |
| 라우트 보호 | 비로그인 시 로그인 화면으로 이동 | 구현됨 |
| 역할 보호 | `leader`, `master` 권한별 관리자 화면 분기 | 구현됨 |

```typescript
UserProfile {
  id: string;
  name: string;
  role: 'master' | 'leader' | 'member';
  status: 'pending' | 'active';
  districtId: string;
  districtName: string;
}
```

| 중보기도 섹션 | 공유된 기도제목과 참여 수 표시 | 구현됨 |

## 3. 성경공부

### 3.1 목록

**페이지:** `/bible-study`

| 기능 | 설명 | 상태 |
|------|------|------|
| 목록 조회 | 주차, 제목, 구절, 완료 여부 표시 | 구현됨 |
| 완료 배지 | 제출 완료 시 표시 | 구현됨 |
| 상세 이동 | 카드 클릭으로 상세 이동 | 구현됨 |
| 발행 자료 노출 | `published=true` 자료만 구역원에게 노출 | 구현됨 |

### 3.2 상세 및 답변

**페이지:** `/bible-study/:id`

| 기능 | 설명 | 상태 |
|------|------|------|
| 공부 내용 표시 | 제목, 본문, 도입, 질문 표시 | 구현됨 |
| 답변 입력 | 질문별 textarea 입력 | 구현됨 |
| 임시 저장 | 작성 중 내용 저장 | 구현됨 |
| 완료 저장 | 완료 상태 반영 | 구현됨 |
| 수정 저장 | 완료 후 재저장 | 구현됨 |
| 기존 답변 로드 | 기존 입력값 자동 복원 | 구현됨 |
| 주간 마감 반영 | 잠금 상태일 때 제출 제한 | 구현됨 |

```typescript
BibleStudy {
  id: string;
  date: string;
  weekNumber: number;
  title: string;
  scripture: string;
  introduction: string;
  questions: string[];
  published: boolean;
}

StudyAnswer {
  id: string;
  studyId: string;
  userId: string;
  answers: Record<number, string>;
  completed: boolean;
  updatedAt: string;
}
```

## 4. 기도제목

**페이지:** `/prayer-requests`, `/prayer-requests/:id`

| 기능 | 설명 | 상태 |
|------|------|------|
| 목록 조회 | 구역 단위 기도제목 표시 | 구현됨 |

| 상세 조회 | 개별 기도제목 상세 화면 | 구현됨 |
| 응답 처리 | 기도 응답 입력 및 수정 | 구현됨 |
| 응답 배지 | 응답 완료 상태 구분 | 구현됨 |
| 구역장 공유 | 리더 공유 여부 설정 | 구현됨 |
| 중보기도 공유 | 구역 단위 공유 전환 | 구현됨 |
| 중보기도 참여 | 다른 사용자의 기도제목에 참여 표시 | 구현됨 |


## 6. 일정 및 출석

**페이지:** `/schedule`

| 기능 | 설명 | 역할 제한 | 상태 |
|------|------|----------|------|
| 일정 조회 | 예정/지난 일정 탭 분리 | 전체 | 구현됨 |
| 일정 등록/수정/삭제 | 일정 CRUD | 구역장 | 구현됨 |
| 출석 응답 | 참석/불참 선택 | 전체 | 구현됨 |
| 출석 현황 조회 | 참석자/불참자 목록 확인 | 구역장 | 구현됨 |
| 첨부 URL 입력 | 일정에 URL 첨부 | 구역장 | 구현됨 |
| 파일 업로드 | Storage 업로드 후 공개 URL 연결 | 구역장 | 구현됨 |

```typescript
Schedule {
  id: string;
  title: string;
  date: string;
  time: string;
  location: string;
  memo: string;
  attachment: string;
  attendanceCheck: boolean;
  createdBy: string;
  createdAt: string;
}

Attendance {
  scheduleId: string;
  userId: string;
  userName: string;
  status: 'attending' | 'absent' | 'pending';
  updatedAt: string;
}
```

## 9. 자동화 기능

| 기능 | 설명 | 상태 |
|------|------|------|
| 오늘의 묵상 수집 | 외부 페이지 스크래핑 + OpenAI 요약 + DB 저장 | 코드 구현됨 |
| 주간 마감 | DB 함수 기반 주간 보고 집계 및 잠금 | 코드 구현됨 |
| 주보 PDF 파싱 | PDF 다운로드 + OpenAI 파싱 + 성경공부 초안 생성 | 코드 구현됨 |

주의:

- 위 자동화 항목은 저장소 기준 코드가 존재한다.
- 실제 운영에서 스케줄이 등록되어 실행 중인지는 별도 확인이 필요하다.

## 10. 현재 운영상 핵심 부족분

- `pg_cron`, Edge Function, Storage 버킷이 운영 환경에서 실제 활성화됐는지 확인이 필요하다.
- PWA/오프라인은 아직 실제 적용 상태로 보기 어렵다.
- 관리자 화면 성능 최적화 여지가 있다.
- 일부 문서는 최근 구현 상태에 맞춰 지속적으로 갱신해야 한다.
