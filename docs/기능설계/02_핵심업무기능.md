# 핵심 업무 기능

## 1. 인증

**페이지:** `/`

| 기능 | 설명 | 상태 |
|------|------|------|
| 로그인 | 이름/비밀번호 입력 후 로컬 사용자 저장 | 구현됨 |
| 회원가입 요청 | 신청 폼 표시 | UI만 |
| 비밀번호 재설정 | 재설정 폼 표시 | UI만 |
| 로그아웃 | 사용자 정보 삭제 후 로그인 페이지 이동 | 구현됨 |
| 라우트 보호 | 비로그인 시 로그인 화면으로 이동 | 구현됨 |

```typescript
User {
  id: string;
  name: string;
  role: 'leader' | 'member';
}
```

## 2. 대시보드

**페이지:** `/dashboard`

| 기능 | 설명 | 상태 |
|------|------|------|
| 통계 카드 | 공부 완료, 성경 읽기, 기도 제목, 현재 주차 | 구현됨 |
| 오늘의 QT | 외부 링크 제공 | 구현됨 |
| 최신 공부 미리보기 | 최근 성경 공부 카드 노출 | 구현됨 |
| 일정 팝업 | 당일 1회만 팝업 표시 | 구현됨 |

## 3. 성경 공부

### 3.1 목록

**페이지:** `/bible-study`

| 기능 | 설명 | 상태 |
|------|------|------|
| 목록 조회 | 주차, 제목, 구절, 완료 여부 표시 | 구현됨 |
| 완료 배지 | 제출 완료 시 표시 | 구현됨 |
| 상세 이동 | 카드 클릭으로 상세 이동 | 구현됨 |
| 자료 공급 방식 | 현재 3건 mock 데이터 | 구현됨 |

### 3.2 상세 및 답변

**페이지:** `/bible-study/:id`

| 기능 | 설명 | 상태 |
|------|------|------|
| 공부 내용 표시 | 제목, 본문, 도입, 질문 표시 | 구현됨 |
| 답변 입력 | 6개 질문 textarea 입력 | 구현됨 |
| 답변 저장 | 사용자별 localStorage 저장 | 구현됨 |
| 완료 처리 | 제출 시 완료 상태 반영 | 구현됨 |
| 기존 답변 로드 | 기존 입력값 자동 복원 | 구현됨 |

```typescript
BibleStudy {
  id: string;
  date: string;
  weekNumber: number;
  title: string;
  scripture: string;
  introduction: string;
  questions: string[];
}

StudyAnswer {
  studyId: string;
  answers: Record<number, string>;
  completed: boolean;
}
```

## 4. 기도 제목

**페이지:** `/prayer-requests`

| 기능 | 설명 | 상태 |
|------|------|------|
| 목록 조회 | 전체 기도 제목 표시 | 구현됨 |
| 등록 | 새 기도 제목 저장 | 구현됨 |
| 응답 처리 | 구역장이 응답 입력 가능 | 구현됨 |
| 응답 배지 | 응답 완료 상태 구분 | 구현됨 |

```typescript
PrayerRequest {
  id: string;
  userId: string;
  userName: string;
  content: string;
  response: string;
  answered: boolean;
  createdAt: string;
  updatedAt: string;
}
```

## 5. 성경 읽기

**페이지:** `/bible-reading`

| 기능 | 설명 | 상태 |
|------|------|------|
| 일일 장수 입력 | 날짜별 읽은 장 수 기록 | 구현됨 |
| 전체 진도 시각화 | 1,189장 기준 프로그레스 | 구현됨 |
| 누적 기록 조회 | 주차별 기록 이력 표시 | 구현됨 |
| 연간 집계 | 2026년 누적 현황 표시 | 구현됨 |

```typescript
BibleReadingLog {
  id: string;
  userId: string;
  date: string;
  chapters: number;
}
```

## 6. 일정 및 출석

**페이지:** `/schedule`

| 기능 | 설명 | 역할 제한 | 상태 |
|------|------|----------|------|
| 일정 조회 | 예정/지난 일정 탭 분리 | 전체 | 구현됨 |
| 일정 등록/수정/삭제 | 일정 CRUD | 구역장 | 구현됨 |
| 출석 응답 | 참석/불참 선택 | 전체 | 구현됨 |
| 출석 현황 조회 | 참석자/불참자 목록 확인 | 구역장 | 구현됨 |

```typescript
Schedule {
  id: string;
  title: string;
  date: string;
  time: string;
  location: string;
  memo: string;
  attendanceCheck: boolean;
  attachment?: string;
  createdBy: string;
  createdAt: string;
}

Attendance {
  scheduleId: string;
  userId: string;
  userName: string;
  status: 'attending' | 'absent' | 'pending';
  updatedAt: string;
}
```

**페이지:** `/admin`

| 탭 | 기능 | 상태 |

## 8. 현재 운영상 핵심 부족분

- 인증이 실사용 수준이 아니다.
- 성경 공부 자료가 하드코딩되어 있다.
- 관리자 운영 기능이 일부 UI 수준에 머물러 있다.
- 파일 업로드, 내보내기, 알림, 통계는 아직 미완성이다.
