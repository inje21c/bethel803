QT 탭 리디자인 시안

현재 흐름 오늘의 QT → 깊은 묵상(AI 4단계) → 기도하기 → 완료·캘린더를 유지하면서, 단계 UI 통일 · 톤 일관화 · 타입/여백을 홈과 같은 기준으로 정리했습니다. 대상 파일: QTMain · QTDeepMeditation · QTPray · QTComplete

진단 1 · 톤 불일치
완료·캘린더 화면만 초록 위주 → 나머지 네이비와 단절. 네이비+골드 + success는 완료에만으로 통일.
진단 2 · 완료 상태 혼동
'묵상 완료'와 'QT 완료'가 분리돼 헷갈림. → QT 완료 = 단일 기준, 깊은 묵상은 선택 심화로 명확히.
진단 3 · 단계 UI 분산
4단계 화면마다 버튼 위치·라벨이 제각각. → 스텝 헤더·진행바·하단 버튼 통일.
유지 · 강점
AI 깊은 묵상 4단계 · 오디오 해설 · 구역 기도제목 연동은 좋은 차별점. 살리고 다듬음.

리디자인 원칙

01 · 톤 통일

네이비+골드 기본. success(초록)는 '완료'에만. 캘린더 완료칸도 네이비로.

02 · 완료 = 단일

'QT 완료' 하나가 진실. 깊은 묵상은 선택 심화로 분리해 혼동 제거.

03 · 단계 통일

4단계 진행바·라벨 상단 고정, 하단 [‹ 이전·다음·건너뛰기] 위치 고정.

04 · 홈과 한 시스템

본문 15·라벨 13·마이크로 12, 거터 단일, 카드인카드 제거 — 홈 기준 동일 적용.
