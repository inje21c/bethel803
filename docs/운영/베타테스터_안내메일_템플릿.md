# 베타 테스터 안내 메일 템플릿 (재사용)

목적: 베타 테스트에 참여한 사용자가 만든 테스트 교회/계정/데이터를 정리(삭제)할 때, 마스터에게 보내는 감사 + 삭제 안내 메일.
배경: 베타 테스터를 활용한 테스트를 **주기적으로** 진행한다 → 라운드가 끝날 때마다 이 템플릿으로 안내 후 정리.
톤 규칙: 공동체·따뜻함 중심. SaaS 마케팅 어조("무료 체험", "지금 시작" 등) 금지. (메모리 feedback-tone-religious-app)

다름이 아니라, 그동안 테스트로 만들어 주신 **「[교회명/모임명]」 공간**을 곧 정리(삭제)하려고 합니다. 베타 기간 동안 시험 삼아 개설된 공간이라, 등록해 주신 내용과 계정 정보는 함께 삭제될 예정입니다.

## 사용 메모
- `[이름]`, `[교회명/모임명]`, `[날짜]` 치환.
- 본인 테스트 계정(예: "테스트교회관리자")은 발송 생략 가능. 실제 지인/외부 테스터에게만 발송.
- 발송은 수동(또는 Gmail 초안). 임의 자동 발송하지 않음.
- 삭제 절차는 설계 문서 10장 참고: deleted_at 설정 → hard_delete_church_superadmin RPC(CASCADE) → auth.users 별도 Edge Function 삭제.

## 2026-06-22 라운드 대상 (베타 오픈 첫 라운드)

|---|---|---|
| 테스트교회 (UUID-T) | 테스트교회관리자 | 본인 테스트 → 생략 가능 |
