## A1. 카카오톡 공지 메시지 생성

### A1.1 개요

구역원과의 소통 채널이 카카오톡 팀채팅이므로, 구역장이 서비스 내에서 공지 메시지를 생성하고 링크와 함께 복사하여 카카오톡에 붙여넣기 할 수 있는 기능.

### A1.2 기능 상세

| 기능 | 설명 |
|------|------|
| 공지 템플릿 선택 | 사전 정의된 공지 유형 선택 |
| 메시지 미리보기 | 생성된 메시지 편집 가능한 텍스트로 표시 |
| 딥링크 자동 삽입 | 해당 기능 페이지 직접 링크 자동 포함 |
| 클립보드 복사 | 버튼 클릭 시 메시지 전체 복사 |

### A1.3 공지 템플릿 유형

```
[구역예배 출석 등록]
📋 {제목} 구역예배 출석 여부를 등록해 주세요.
👉 {서비스 URL}/schedule
마감: {날짜} {시간}

[성경공부 답변 작성]
📖 이번 주 구역성경공부 답변을 작성해 주세요.
👉 {서비스 URL}/bible-study/{id}
마감: {주일 오전 11:20}

[성경 읽기 장수 입력]
📌 이번 주 성경 읽기 장수를 입력해 주세요.
👉 {서비스 URL}/bible-reading
마감: {주일 오전 11:20}

[기도 제목 등록]
🙏 이번 주 기도 제목을 등록해 주세요.
👉 {서비스 URL}/prayer-requests

[사용자 정의]
구역장이 직접 입력
```

## A2. 성경 공부 답안 접근 권한

### A2.1 변경 내용

현재: 답변이 localStorage에 저장되며 접근 제한 없음
변경: **답변 내용은 작성자 본인만 열람 가능. 구역장 포함 타인은 완료 여부(O/X)만 확인.**

### A2.2 적용 규칙

| 대상 | 답변 내용 | 완료 여부 |
|------|----------|----------|
| 본인 | 열람 + 편집 가능 | 표시 |
| 구역원 (타인) | 열람 불가 | 표시 |
| 구역장 | 열람 불가 | 표시 |

### A2.3 변경 대상 파일

- [BibleStudyDetail.tsx](../src/pages/BibleStudyDetail.tsx) — 답변 로드 시 `userId` 일치 여부 검증
- [AdminDashboard.tsx](../src/pages/AdminDashboard.tsx) — 공부 관리 탭에서 답변 내용 컬럼 제거, 완료 여부만 표시
- [store.ts](../src/lib/store.ts) — `getAnswers()` 함수에 userId 필터링 추가

### A2.4 구현 방식

```typescript
// store.ts - 본인 답변만 반환
getMyAnswer(studyId: string, userId: string): StudyAnswer | null {
  const all = getAnswers();
  return all.find(a => a.studyId === studyId && a.userId === userId) ?? null;
}

// 완료 여부만 반환 (구역장용)
getCompletionStatus(studyId: string): { userId: string; completed: boolean }[] {
  return getAnswers()
    .filter(a => a.studyId === studyId)
    .map(a => ({ userId: a.userId, completed: a.completed }));
}
```

### A2.5 StudyAnswer 모델 변경

```typescript
// 기존

  studyId: string;

  completed: boolean;

// 변경 (userId 추가)
```

## A3. 오늘의 묵상 자동 스크래핑 + AI 요약

### A3.1 개요

매일 오전 6시에 `https://sum.su.or.kr:8888/bible/today` 페이지를 스크래핑하여 당일 묵상 내용을 정리하고, OpenAI를 이용해 해설 탭 내용을 요약하여 대시보드에 표시.

### A3.2 수집 데이터

| 항목 | 위치 | 표시 위치 |
|------|------|----------|
| 오늘의 성경 구절 | 페이지 우측 성경 본문 | 우측 패널 |
| 해설 탭 내용 | "해설" 탭 텍스트 | OpenAI 요약 후 표시 |
| 오디오 해설 링크 | 페이지 하단 오디오 플레이어 | 요약 하단 링크 |
| 매일성경 오늘의 묵상 링크 | 페이지 하단 | 요약 하단 링크 |

### A3.3 스크래핑 스케줄

```
매일 오전 06:00 (KST) 자동 실행
→ 스크래핑 실패 시 오전 06:30 1회 재시도
→ 당일 데이터 DB 또는 캐시 저장
→ 다음날 오전 06:00 갱신
```

### A3.4 OpenAI 요약 프롬프트

```
역할: 교회 구역원을 위한 매일성경 묵상 도우미
입력: {해설 탭 전문}
출력 조건:
  - 핵심 메시지 2~3문장 요약
  - 오늘의 적용 질문 1가지
  - 한국어, 친근하고 쉬운 표현
  - 300자 이내
```

**사용 모델:** `gpt-4o-mini` (비용 효율)

### A3.5 대시보드 UI 구성

```
[오늘의 묵상 카드]
┌─────────────────────────────────────────────┐
│  오늘의 묵상          {날짜}                  │
├──────────────────┬──────────────────────────┤
│  AI 요약 내용    │  오늘의 성경 구절          │
│  (3~4줄)         │  (본문 텍스트)             │
│                  │                            │
│  적용 질문:      │                            │
│  "..."           │                            │
├──────────────────┴──────────────────────────┤
│  [오디오 해설 듣기]  [매일성경 오늘의 묵상]   │
└─────────────────────────────────────────────┘
```

### A3.6 데이터 모델

```typescript

  id: string;
  date: string;                  // YYYY-MM-DD
```

## A4. 구역성경공부 주보 PDF 자동 수집

### A4.1 개요

벧엘교회 일산 메인 홈페이지에 매주 주일 게시되는 주보 PDF의 7페이지에서 구역성경공부 내용을 자동으로 추출하여 성경 공부 자료로 등록.

### A4.2 성경공부 주기 및 타이밍

```
주보 게시:    매주 주일 (당일)
성경공부 시작: 주보 게시 당일 주일부터
성경공부 마감: 다음 주일 오전 11:20 (구역예배 시작 전)

예시)
  3월 9일(주일)  주보 게시 → 3월 9일~15일 성경공부 가능
  3월 16일(주일) 오전 11:20 → 3월 9일 주보 공부 마감
  3월 16일(주일) 주보 게시 → 3월 16일~22일 성경공부 시작
```

### A4.3 스크래핑 스케줄

```
매주 일요일 저녁 20:00 (KST) 자동 실행
→ 주보 PDF 다운로드
→ 7페이지 텍스트 추출
→ OpenAI로 성경공부 구조화 파싱
→ 성경 공부 자료 DB 자동 등록 (비발행 상태)
→ 구역장에게 검토 알림 발송
→ 구역장 승인 후 발행 (또는 자동 발행 설정)
```

### A4.4 OpenAI 파싱 프롬프트

```
역할: 교회 주보 PDF에서 구역성경공부 내용을 추출하는 파서
입력: {PDF 7페이지 추출 텍스트}
출력 형식 (JSON):
{
  "title": "이번 주 공부 제목",
  "scripture": "성경 구절 (예: 요한복음 3:1-16)",
  "introduction": "도입 내용 또는 배경 설명",
  "questions": [
    "첫 번째 질문",
    "두 번째 질문",
    ...
  ]
}
```

**사용 모델:** `gpt-4o` (PDF 파싱 정확도 우선)

### A4.5 데이터 모델 변경

```typescript
BibleStudy {
  id: string;
  weekNumber: number;
  studyPeriodStart: string;   // 주보 게시일 (주일)
  studyPeriodEnd: string;     // 마감일 (다음 주일 11:20)
  title: string;
  scripture: string;
  introduction: string;
  questions: string[];
  sourcePdfUrl: string;       // 원본 주보 PDF URL
  published: boolean;         // 발행 여부
  publishedAt: string;
  createdAt: string;
}
```

### A4.6 마감 처리 자동화

```
매주 주일 오전 11:20 (KST) 자동 실행:
  1. 현재 주차 성경공부 마감 처리 (isLocked = true)
  2. 성경 읽기 장수 주간 합산 스냅샷 저장
  3. 기도 제목 해당 주차 마감 플래그 설정
  4. 구역장에게 "마감 완료" 알림
```

### A4.7 백엔드 요구사항

```
GET  /api/studies                    - 공부 목록
GET  /api/studies/:id                - 공부 상세
POST /api/studies/scrape             - 수동 스크래핑 트리거
PATCH /api/studies/:id/publish       - 발행 처리 (구역장)
PATCH /api/studies/:id/close         - 마감 처리 (자동/수동)

CRON: 매주 일요일 20:00 → PDF 스크래핑
CRON: 매주 일요일 11:20 → 마감 처리
```
