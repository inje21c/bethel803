구역 탭 리디자인 시안

상단 4탭(성경공부·기도제목·일정·성경읽기) + 기도제목 내부 2탭 + 성경읽기 내부 4탭. 2단계 탭 구조의 정보 위계를 정리하고 역할 분기·데이터 연결(홈 출석약속)·타입·여백을 통일합니다.

진단 1 · 서브탭 줄바꿈
CommunitySubNav 아이콘+텍스트 4개가 360px에서 줄바꿈. → nowrap + overflow-x-auto, 패딩 축소.
진단 2 · 2단계 탭 혼잡
성경읽기 내 4탭이 상위 탭과 구분 안 됨. → 상위=세그먼트(pill), 하위=라인탭으로 시각 레벨 분리.
진단 3 · 역할 미노출
구역장 전용 기능(일정 등록·출석 편집·성경공부 등록)이 맥락 없이 나타남. → 구역장 배지·역할 라벨로 명시.
유지 · 강점
중보기도 Heart 토글, 참석 RSVP(홈 출석약속 소스), 성경 이어읽기(localStorage), 읽기표+기록 — 모두 살리고 다듬음.

구역 탭 리디자인 원칙

01 · 탭 레벨 분리

상위 = 세그먼트(pill), 하위 = 라인탭. 각 레벨이 시각적으로 구분돼 혼잡 제거.

02 · 데이터 연결

일정 RSVP → 홈 출석약속, 성경읽기 완료 → 홈 나의성경읽기 진도. 단일 소스 원칙.

03 · 역할 명시

구역장 전용 기능에 [구역장] 배지 명시. 구역원에겐 보이지 않는 버튼.

04 · 홈과 한 시스템

타입·여백·아이콘 기준 동일. success는 완료에만. 이어읽기 배너는 골드 톤.
